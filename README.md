# Opal Devtools

Currently provides a console, available in the Opal tab of Chrome Devtools.

- clone
- yarn install
- bundle install
- yarn run production_build

and in chrome extensions, developer mode, load the `extension` directory.

#### Using
- opal
- opal-browser
- opal-webpack-loader
- isomorfeus-redux
- isomorfeus-react
- OpalConsole, integrated.
